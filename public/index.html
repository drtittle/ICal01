Welcome to WebRTC on Bluemix!

You're entering a virtual Para-military operator control center.
This example addresses many use cases:

  1. Military style operators wearing camera enabled goggles so base commanders can see real time action.
  2. Police and public service folks with body cams recording life in real time.
  3. Medical personnel examining patients via a home computer or smart device.
  4. More...
This uses a mesh networking model to implement many-to-many video/data connection.

  1. All users can watch each other; all of them can speak as well.
  2. All users can exchange chat messages.<
  3. All users can share files as well. If you share a file, all others will receive it.
